# Platform Roadmap

This document outlines upcoming capabilities and releases planned for the FlientSec platform.

---

## 1. Milestones Overview

### Phase 1: Go Agent Core (v0.1.0-alpha - Current)
- Automated local system checker checks.
- YAML rules policy local evaluation engine.
- PostgreSQL FastAPI API backend services.
- Light/Dark Next.js administration dashboard.

### Phase 2: Agent Expansion (v0.2.0-beta)
- Cross-platform agent clients (macOS, Windows).
- Slack, Microsoft Teams, and email compliance alert notifications.
- Remote policy synchronization triggers.

### Phase 3: Enterprise Trust (v1.0.0-release)
- SAML/OIDC Single Sign-On integration.
- Custom script-based compliance checks.
- Audit history export logs (CSV/JSON/Syslog).
