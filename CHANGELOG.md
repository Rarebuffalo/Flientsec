# Changelog

All notable changes to the FlientSec project will be documented in this file. This project adheres to Semantic Versioning.

## [v0.1.0-alpha] - 2026-07-12

### Added
- **Lightweight Go Agent Daemon:**
  - Automated system checking units (Firewall rules, Disk Encryption, Package Upgrades, OpenSSH socket limits, Compiler toolchains).
  - YAML policy evaluation score computation engines.
  - Telemetry retry buffers caching check-ins locally during network outages.
- **FastAPI Backend Server:**
  - Seeds Postgres schemas with user accounts and yaml policies on first run.
  - Relational mapping schemas mapping connected hosts, findings history, and check runs.
- **Next.js Frontend & Marketing Home:**
  - Unified Light/Dark theme setups matching Tailwind system specifications.
  - Restructured routes layout inside route groups.
  - Zoomed interactive dual-pane SaaS hero dashboard card mockup.
  - Fully visual node Problem comparison flows.
  - YAML technical proof checklist comparisons.
  - Objections FAQs mapping Microsoft Intune/Jamf MDM alternatives.
